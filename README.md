# Academic Pages profile update for Yeonwoo Cha

This package rewrites the provided personal webpage into a native `academicpages` structure.

## Files
- `CONFIG_SNIPPET.yml` → paste/merge into your template `_config.yml`
- `_data/navigation.yml` → replace your navigation file
- `_pages/about.md` → home page
- `_pages/education.md` → education page
- `_pages/contact.md` → contact page
- `_pages/publications.html` → publications archive page
- `_publications/*.md` → individual publication entries

## One-time manual step
Put your profile photo at:

`images/profile.jpeg`

If you prefer another filename, update `author.avatar` in `CONFIG_SNIPPET.yml`.

## Important
The `url` and `repository` fields assume your GitHub Pages repo is:
- URL: `https://yeonwoo378.github.io`
- Repository: `yeonwoo378/yeonwoo378.github.io`

If your actual repo differs, update those two values.
